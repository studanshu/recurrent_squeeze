# recurrent_squeeze

## Dataset
CIFAR - 10

## Requirements
* python3
* pytorch
* numpy
* scipy
* matplotlib

## How to run?
There are 3 different models:
* SqueezeNet ```python3 basic_squeeze.py```
* Recurrent Squeeze and Excitation SqueezeNet ```python3 recurrent_squeeze.py```
* Squeeze and Excitation SqueezeNet ```python3 se_squeeze.py```
